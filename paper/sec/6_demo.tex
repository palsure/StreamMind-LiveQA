\section*{System Demonstration}
\label{sec:demo}

We provide a live web demo that lets users interact with \ours through a browser (see the supplementary material for a screenshot).
A FastAPI backend runs the full CLIP + BLIP + Flan-T5 pipeline and communicates with the front end over WebSockets.
The interface splits into two panels: a \emph{video panel} showing the live feed with the SKM filmstrip and per-frame importance scores, and a \emph{chat panel} accepting free-form questions with answers annotated by temporal scope, context frame count, and per-query latency.
Users can stream from their webcam, upload a local video, or choose from pre-loaded sample clips.

A dedicated \emph{workshop presentation mode} enlarges all UI elements for projector visibility and offers one-click sample questions stepping through all three temporal scopes.
The demo runs on GPU ($\sim$250\,ms per query on A100) or CPU (1--3\,s) and ships with a Docker configuration for one-command startup (\texttt{docker compose up}).
All source code, the LiveQA-Bench dataset, evaluation scripts, and the interactive demo are publicly available at \url{https://github.com/palsure/StreamMind-LiveQA}.
